标题：客服接入概述（微信客服 API）

================================================================================
客服接入概述
================================================================================

企业可获取对应的客服链接，在视频号、微信搜一搜、网页等微信内和微信外场景接入微信客服，各场景的用户点击后即可向对应的微信客服发起咨询。

各场景的接入方式（官方目录要点）：

1、在微信内接入（细节见官网该章节）

2、在微信外接入（官方提供运营/活动页入口，链接可能随官方活动更新，以下仅为归档时抓到的参考链接）
   - 在 App 中接入：见 https://work.weixin.qq.com/nl/act/p/a9ba0dc512144b62 及 SDK 说明（以官网最新为准）
   - 在微信外网页接入：见 https://work.weixin.qq.com/nl/act/p/4030a5b69149404d（以官网最新为准）
