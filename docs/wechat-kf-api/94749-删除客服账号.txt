标题：删除客服账号（微信客服 API）

================================================================================
删除客服账号
================================================================================

删除已有的客服账号。

请求方式：POST（HTTPS）
请求地址：https://qyapi.weixin.qq.com/cgi-bin/kf/account/del?access_token=ACCESS_TOKEN

请求示例（JSON）：
{
    "open_kfid": "wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyJTw"
}

参数说明：
  access_token  必填  string  调用接口凭证
  open_kfid     必填  string  客服账号 ID，不多于 64 字节

权限说明：仅允许微信客服的 Secret 所获取的 access_token 调用。

返回结果示例（JSON）：
{
    "errcode": 0,
    "errmsg": "ok"
}

返回字段说明：
  errcode   int32   返回码
  errmsg    string  错误码描述
