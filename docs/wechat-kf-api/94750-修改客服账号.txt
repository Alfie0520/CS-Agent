标题：修改客服账号（微信客服 API）

================================================================================
修改客服账号
================================================================================

修改已有的客服账号，可修改客服名称和头像。

请求方式：POST（HTTPS）
请求地址：https://qyapi.weixin.qq.com/cgi-bin/kf/account/update?access_token=ACCESS_TOKEN

请求示例（JSON）：
{
    "open_kfid": "wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyJTw",
    "name": "修改客服名",
    "media_id": "294DpAog3YA5b9rTK4PjjfRfYLO0L5qpDHAJIzhhQ2jAEWjb9i661Q4lk8oFnPtmj"
}

参数说明：
  access_token  必填  string  调用接口凭证
  open_kfid     必填  string  要修改的客服账号 ID，不多于 64 字节
  name          否    string  新的客服账号名称；如不需要修改可不填，不多于 16 个字符
  media_id      否    string  新的客服头像临时素材；如不需要修改可不填（上传临时素材接口获取），不多于 128 个字节

权限说明：仅允许微信客服的 Secret 所获取的 access_token 调用。

返回结果示例（JSON）：
{
    "errcode": 0,
    "errmsg": "ok"
}

返回字段说明：
  errcode   int32   返回码
  errmsg    string  错误码描述
